# Zamin

پروژه Zamin یک framework چندبخشی در اکوسیستم .NET است که با تمرکز بر معماری تمیز، DDD، CQRS و توسعه‌پذیری تیمی شکل گرفته است.

این پروژه بخشی از اکوسیستم **ARO-Mind** است و با هدف فراهم کردن یک بستر قابل اتکا برای رشد سایر سیستم‌ها و محصولات نرم‌افزاری توسعه داده می‌شود.

## وضعیت فعلی

در حال حاضر Zamin در مسیر مهاجرت از .NET 9 به .NET 10 قرار دارد و مستندات معماری، ساختار پروژه و migration آن به‌صورت مرحله‌ای در حال تکمیل است.
